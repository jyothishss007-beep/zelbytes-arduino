# Task 4: Analog Sensor Calibration Suite

## Objective

Calibrate and log data from:

- LDR
- DHT22
- HC-SR04 Ultrasonic Sensor
- Soil Moisture Sensor

## CSV Output Format

millis,temp,humidity,distance_cm,soil_raw

## Sensors Used

- LDR
- DHT22
- HC-SR04
- Capacitive Soil Moisture Sensor v1.2

## Board

Arduino Uno

Successfully calibrated and tested all sensors individually and through a unified logger producing CSV-formatted serial output.
